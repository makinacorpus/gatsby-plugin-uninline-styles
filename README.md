## Description

This plugin allow to un-inline CSS from generated HTML files

## How to install

`npm install gatsby-plugin-uninline-styles`

## How to use

```javascript
// In your gatsby-config.js
module.exports = {
  plugins: [
    'gatsby-plugin-uninline-styles',
  ],
};
```
